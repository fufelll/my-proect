
## Список использованных команд
1. `git clone` — клонирование репозитория
2. `git add` — добавление файлов
3. `git commit` — фиксация изменений
4. `git push` — отправка на удалённый репозиторий
5. `git branch` / `git checkout` — работа с ветками
6. `git merge` — слияние веток
7. `git tag` — создание меток
8. `git reset` / `git checkout <hash>` — откат версий
9. `git log` — просмотр истории
